Instalação do Drupal com Docker

Passo 1: Baixar as Imagens Docker

sudo docker pull mysql
sudo docker pull drupal

Passo 2: Executar os Contêineres

sudo docker run --name drupal -p 8080:80 -d drupal
sudo docker run --name mysql-drupal -e MYSQL_ROOT_PASSWORD=rootpass -e MYSQL_DATABASE=drupaldb -e MYSQL_USER=drupaluser -e MYSQL_PASSWORD=drupalpass -p 3306:3306 -d mysql

Passo 3: Configurar a Rede Docker

sudo docker network create drupalnet
sudo docker network connect drupalnet mysql
sudo docker network connect drupalnet drupal

Passo 4: Inspecionar a Rede

Verifique se a rede foi configurada corretamente:

sudo docker network inspect drupalnet

Passo 5: Verificar Conexão com o Banco de Dados

sudo docker exec -it drupal bash
apt update && apt install -y default-mysql-client
mysql -hmysql -udrupaluser -pdrupalpass drupaldb

Se conectar ao banco de dados, a configuração está correta.
