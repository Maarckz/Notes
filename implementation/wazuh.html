Instalação do Wazuh com Assistente

Pré-requisitos

  • Sistema Operacional: 64-bit (x86_64/AMD64).
  • Recursos Mínimos:
    • CPU: 2 núcleos.
    • RAM: 4 GB.
    • Disco: 100 GB (recomendado 200 GB para produção).
  • Acesso Root: Permissões de superusuário (sudo).
  • Portas Liberadas:
    • 9200 (Wazuh Indexer).
    • 55000 (Wazuh Manager).
    • 443 (Wazuh Dashboard).
  • Dependências Instaladas:
sudo apt-get install curl tar -y  # Ubuntu/Debian
    # ou
    sudo yum install curl tar -y      # CentOS/RHEL

Passo 1: Executar o Script de Instalação

O comando abaixo baixa o script e inicia a instalação automática (-a = all-in-one):

curl -sO https://packages.wazuh.com/4.12/wazuh-install.sh && sudo bash ./wazuh-install.sh -a

O que o script faz:

  • Configura o Wazuh Indexer (armazenamento de dados).
  • Instala o Wazuh Manager (servidor central).
  • Instala o Wazuh Dashboard (interface web).
  • Gera certificados SSL automáticos para comunicação segura.
  • Define credenciais padrão:
    • Usuário do Dashboard: admin.
    • Senha do Dashboard: admin.

Passo 2: Verificar a Instalação

A. Verificar Serviços em Execução

sudo systemctl status wazuh-indexer wazuh-manager wazuh-dashboard

Todos devem estar active (running).

B. Acessar o Wazuh Dashboard

  • URL: https://<IP_DA_SUA_MAQUINA>.
  • Credenciais Iniciais:
    • Usuário: admin.
    • Senha: admin.
