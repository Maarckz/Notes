Instalar FirewallD

sudo apt update && sudo apt install firewalld


Iniciar e Habilitar FirewallD

sudo systemctl start firewalld

sudo systemctl enable firewalld


Verificar Status do FirewallD

sudo firewall-cmd --state


Adicionar Serviços e Portas

Adicionar Serviço HTTP

sudo firewall-cmd --permanent --add-service=http


Adicionar Serviço HTTPS

sudo firewall-cmd --permanent --add-service=https


Adicionar Porta SSH

sudo firewall-cmd --permanent --add-port=22/tcp


Remover Serviços e Portas

Remover Serviço FTP

sudo firewall-cmd --permanent --remove-service=ftp


Remover Porta 21

sudo firewall-cmd --permanent --remove-port=21/tcp


Recarregar Configurações do FirewallD

sudo firewall-cmd --reload


Listar Serviços e Portas Ativos

sudo firewall-cmd --list-all


Configurar Zona do Firewall

Definir Zona Pública como Padrão

sudo firewall-cmd --set-default-zone=public


Adicionar IP à Lista de Confiança

sudo firewall-cmd --permanent --zone=trusted --add-source=192.168.1.100


Remover IP da Lista de Confiança

sudo firewall-cmd --permanent --zone=trusted --remove-source=192.168.1.100


Habilitar Logging

sudo firewall-cmd --set-log-denied=all


Reiniciar FirewallD

sudo systemctl restart firewalld
